# 2048WinFormsApp
Клон оригинальной игры "2048". Работа с файловой системой и с движением  плиток.

## Человеческий параграф

### Правила игры 
С помощью стрелок нужно двигать плитки и производить слияние одноименных плиток.

### Конец игры
Игра заканчивается когда **пользователь** собирает клетку 2048.

## Техническая часть
• Движение плиток происходит нажатием стелок на клавиатуре
• По возразстаничею чисел в клетке меняется цвет
• Есть функция изменения размера игрового поля

### Сериализация

• Сохранял результаты игры с помощью сериализации и десериализации данных в формате JSON.
